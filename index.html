Description:

A lightweight Python-based automation tool for structural engineers and detailers using Tekla Structures.

This project parses DXF files to extract Columns and Beams data, then dynamically inserts them into Tekla models via the Tekla Open API. Built with modular Python code and designed for scalability, the tool reduces manual input, minimizes human error, and speeds up the modeling workflow by up to 80%.
